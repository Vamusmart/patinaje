# Escuela de Patinaje Artístico sobre Ruedas Madrid - Landing Page

Una landing page moderna, totalmente responsive y altamente optimizada para una escuela de patinaje artístico sobre ruedas en Madrid. El sitio está diseñado para presentar de manera clara y atractiva las clases para niños, jóvenes y adultos, así como la formación técnica, la danza y la preparación para competiciones.

## 🚀 Características del Proyecto

- **Diseño Totalmente Responsive:** Optimizado para ofrecer una experiencia visual excelente en smartphones, tablets y ordenadores de escritorio.
- **Rendimiento & SEO On-Page:**
  - Etiquetas **Open Graph** (`og:title`, `og:image`, `og:description`, etc.) completamente configuradas para una previsualización atractiva al compartir el enlace en redes sociales.
  - Optimización del tiempo de carga (LCP) mediante el `preload` de la tipografía principal y de la primera imagen del carrusel (*hero image*).
  - Uso de formatos de imagen de última generación (`.webp`) junto con el elemento `<picture>` y atributos `srcset`/`sizes` para servir el tamaño de imagen idóneo según el dispositivo.
  - Inclusión de elementos semánticos y clases de accesibilidad (`sr-only`) para mejorar el posicionamiento en buscadores (SEO).
- **Interactividad con JavaScript Nativo (Vanilla JS):**
  - **Slideshow/Carrusel dinámico** con transiciones suaves (`fade`) y controles interactivos (*dots*) para destacar los pilares de la escuela.
  - **Menú de navegación móvil animado** (efecto hamburguesa) para facilitar la usabilidad en pantallas pequeñas.
  - **Formulario de contacto interactivo** con validación de campos en el lado del cliente y sistema de envío integrado a través de la API de FormSubmit.
  - **Banner de cookies dinámico** para cumplir con las normativas vigentes de privacidad y consentimiento del usuario.

## 🛠️ Tecnologías Utilizadas

- **HTML5:** Estructura semántica limpia y accesible (`<header>`, `<main>`, `<section>`, `<picture>`, `<aside>`).
- **CSS3:** Estilos personalizados, diseño adaptativo, transiciones y efectos de animación fluidos sin dependencias de frameworks externos.
- **JavaScript (ES6+):** Programación interactiva pura para la gestión del carrusel, el menú móvil, el formulario de contacto y el pop-up de cookies.
- **Font Awesome (v5.15.3):** Iconos sociales cargados de forma asíncrona para no bloquear el renderizado inicial de la página.

## 💻 Entorno de Desarrollo

- **Editor de código:** Visual Studio Code ([Sitio Oficial](https://code.visualstudio.com/))
- **Estándares:** Código optimizado que cumple con las directrices de validación del W3C para HTML5 y CSS3.

## 📂 Estructura de Secciones de la Página

1. **Header & Navigation:** Logotipo corporativo responsive y menú de navegación rápida mediante enlaces de anclaje interno.
2. **Hero Slideshow:** Banner dinámico de presentación con tres enfoques clave: *Patinaje Artístico*, *Clases Para Todos* y *Competición Y Danza*.
3. **Sección de Tarjetas (Ventajas):** Tres bloques informativos sobre la técnica individual, los programas adaptados por edades y la preparación coreográfica.
4. **Columnas Informativas (Valores):** Espacio visual y descriptivo que profundiza en la pasión, la evolución técnica y la formación integral de los alumnos.
5. **Sección "La Escuela" (Quiénes Somos):** Detalle de la misión de la escuela acompañado de una sección lateral de imagen corporativa de alta calidad.
6. **Formulario de Contacto:** Bloque interactivo para consultas y reservas con datos de contacto directo de la sede en Madrid.
7. **Footer & Cookies:** Enlaces a redes sociales, derechos reservados y un sistema integrado de aceptación de política de cookies.
